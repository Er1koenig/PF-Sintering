Part I — Model Overview

This page summarises a small project with IWM (Institute for Materials Applications in Mechanical Engineering, RWTH Aachen) that reproduces the 2D multi-phase-field model by Wang et al. (DOI: 10.1016/j.actamat.2005.10.032) The model captures solid-state sintering and grain coarsening using the Cahn–Hilliard equation for the density field c, the Allen–Cahn equation for grain-orientation order parameters ηi, and rigid-body translation/rotation of each grain via net inter-grain force and torque. I implemented the variational formulation in MATLAB, ran a multi-grain test case, and visualised densification and pore shrinkage.
